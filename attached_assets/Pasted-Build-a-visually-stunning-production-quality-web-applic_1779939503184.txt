Build a visually stunning, production-quality web application using Python and FastAPI that can accurately decode and translate ancient Devanagari/Sanskrit inscriptions from uploaded images into English.

Core Requirements
Backend must use Python + FastAPI
Users should be able to:
Upload images of ancient inscriptions, manuscripts, temple carvings, copper plates, palm-leaf texts, stone engravings, and monument scripts
Upload low-quality, damaged, noisy, blurred, tilted, faded, or partially broken inscription images
The system should:
Detect ancient Devanagari/Sanskrit text from images
Perform image enhancement and restoration automatically
Remove noise and improve readability using AI-based preprocessing
Extract text using OCR specialized for Sanskrit/Devanagari
Translate extracted Sanskrit/ancient Devanagari text into accurate English
Display:
Original uploaded image
Enhanced/restored image
Extracted Sanskrit text
English translation
Confidence score
Include support for:
Ancient inscriptions on stone monuments
Temple carvings
Historical scripts
Old manuscript scans
Weathered inscriptions
Damaged monument engravings
AI/ML Features

Use advanced AI and computer vision techniques:

OCR models for Sanskrit/Devanagari recognition
Transformer/NLP-based translation models
Image denoising and restoration
Contrast enhancement
Super-resolution for blurry inscription images
Edge sharpening and adaptive thresholding
Perspective correction for angled monument photos
